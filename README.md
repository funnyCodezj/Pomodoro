## 快速开始 / Quick Start

### 开发模式 / Development Mode

```bash
# 终端 1 — 后端 / Terminal 1 — Backend
cd pomodoro
pip install -r requirements.txt
uvicorn backend.main:app --reload --port 8765

# 终端 2 — 前端 / Terminal 2 — Frontend
cd pomodoro/frontend
npm install
npm run dev
```

浏览器打开 `http://localhost:5173` 即可使用。
Then open `http://localhost:5173` in your browser.

### 生产模式（桌面应用）/ Production / Desktop Mode

```bash
cd pomodoro/frontend
npm run build
cd ..
python app.py
```

### 打包 Windows 可执行文件 / Build Windows Executable

```bash
cd pomodoro/frontend && npm run build
cd ..
pyinstaller --name "Pomodoro" --icon pomodoro.ico --onedir --windowed \
  --add-data "frontend/dist;frontend/dist" \
  --hidden-import webview --hidden-import webview.platforms.winforms \
  --hidden-import uvicorn --hidden-import uvicorn.logging \
  --hidden-import uvicorn.loops.auto --hidden-import uvicorn.protocols.http.auto \
  app.py
# 输出：dist/Pomodoro/Pomodoro.exe
```

## 配置项 / Configuration

设置持久化存储在 SQLite（`pomodoro.db`）中，可通过应用内设置面板调整。
Settings are persisted in SQLite (`pomodoro.db`) and can be adjusted from the in-app settings panel:

| 设置项 / Setting | 默认值 / Default | 说明 / Description |
|---|---|---|
| 工作时长 / Work Duration | 25 min | 专注时长 / Focus session length |
| 短休息 / Short Break | 5 min | 工作间短休 / Short rest between work sessions |
| 长休息 / Long Break | 15 min | 周期末长休 / Extended rest after N cycles |
| 长休息间隔 / Long Break Interval | 4 | 多少个工作后长休息 / Work sessions before a long break |
