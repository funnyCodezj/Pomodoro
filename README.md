# 🍅 番茄钟 (Pomodoro Timer)

## ✨ Features

- **Pomodoro Timer** — Work / Short Break / Long Break cycles with SVG ring progress animation
- **Dark Theme** — Eye-friendly deep purple dark mode UI
- **Statistics** — Weekly bar chart & yearly line chart for completed sessions
- **History** — Browse and delete past sessions
- **Customizable** — Adjust work duration, break durations, and long break interval
- **Desktop Native** — Runs as a standalone desktop window via PyWebView (420×700, fixed size)
- **Portable** — Single-file backend + compiled frontend, can be packaged into a Windows `.exe`

## 🚀 Quick Start

### Development Mode

```bash
# Terminal 1 — Backend
cd pomodoro
pip install -r requirements.txt
uvicorn backend.main:app --reload --port 8765

# Terminal 2 — Frontend
cd pomodoro/frontend
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

### Production / Desktop Mode

```bash
cd pomodoro/frontend
npm run build
cd ..
python app.py
```

### Build Windows Executable

```bash
cd pomodoro/frontend && npm run build
cd ..
pyinstaller --name "Pomodoro" --icon pomodoro.ico --onedir --windowed \
  --add-data "frontend/dist;frontend/dist" \
  --hidden-import webview --hidden-import webview.platforms.winforms \
  --hidden-import uvicorn --hidden-import uvicorn.logging \
  --hidden-import uvicorn.loops.auto --hidden-import uvicorn.protocols.http.auto \
  app.py
# Output: dist/Pomodoro/Pomodoro.exe
```

## ⚙️ Configuration

Settings are persisted in SQLite (`pomodoro.db`) and can be adjusted from the in-app settings panel:

| Setting | Default | Description |
|---|---|---|
| Work Duration | 25 min | Focus session length |
| Short Break | 5 min | Short rest between work sessions |
| Long Break | 15 min | Extended rest after N cycles |
| Long Break Interval | 4 | Number of work sessions before a long break |

## 📊 API Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/api/health` | Health check |
| `POST` | `/api/sessions` | Create a session record |
| `GET` | `/api/sessions` | List sessions (query: `limit`) |
| `DELETE` | `/api/sessions` | Clear all sessions |
| `DELETE` | `/api/sessions/{id}` | Delete one session |
| `GET` | `/api/settings` | Get current settings |
| `PUT` | `/api/settings` | Update settings |
| `GET` | `/api/stats` | Get weekly statistics |
| `GET` | `/api/stats/yearly` | Get yearly statistics (query: `year`) |
